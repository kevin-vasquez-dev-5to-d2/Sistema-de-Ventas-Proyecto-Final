# Sistema de Ventas (Gestion de Categorias, Productos, Clientes, Ventas y Creacion de Reportes)

## Descripción
Este proyecto es una aplicación de escritorio diseñada para gestionar las operaciones comerciales de un negocio. El sistema permite administrar categorías, productos y clientes, además de registrar ventas y generar reportes en base a las ventas ya creadas. 


## Tecnologías Utilizadas
* **Lenguaje:** C#
* **Interfaz:** Windows Forms (.NET Framework)
* **Base de Datos** SQL Server
* **Entorno de Desarrollo:** Visual Studio 2026

## Uso o Ejecución
1.  Abrir el proyecto en Visual Studio.
2.  Darle a "Iniciar" o presionar F5.
3.  Registrar las categorías necesarias.
4.  Agregar productos y clientes.
5.  Realizar una venta desde el módulo correspondiente.
6.  Generar Facturas en base a las ventas registradas.
